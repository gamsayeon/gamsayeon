💻 현재는 주로 Java와 Spring Framework를 활용하여 백엔드 개발에 집중하고 있습니다.

🌱 새로운 기술을 배우고, 프로젝트를 통해 실전 경험을 쌓는 데에 열정을 가지고 있습니다.

📚 구체적인 목표로는 우선 5년 후에는 컨퍼런스 및 모임을 통해 지식을 공유하는 활발한 백엔드 개발자가 되고자 합니다.
